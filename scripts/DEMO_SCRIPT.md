**Say:**

## [0:20–0:50] RUN IT — the Akamai two-tier brain

**Do:** On the billboard card, drag the slider slowly left→right. Pause one beat on the full final. Point at the param chips + the italic rationale line ("why these params" — judges love this).

## One-breath answers (Q&A)
- **How do you use Akamai?** Two-tier LLM on Akamai Inference Cloud: Qwen3-8B does the high-token creative drafting cheap; Qwen3-14B runs once per run to plan Magnific parameters. OpenAI-compatible, so it ports to any Akamai-hosted model.
- **How do you use Magnific?** Four engines: Mystic text-to-image for drafts, Relight with LLM-chosen light direction, the Upscaler with LLM-chosen creativity/factor, background removal for listings — plus the video engine for hero ad spots.
- **What's novel?** The LLM is the art director, not the artist: it emits structured enhancement parameters and a rationale per channel, and the ledger proves the economics per decision.
